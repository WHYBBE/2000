#449 – Data Binding Elements in a Collection to a Grid, part II

2011-12-13 | Original URL | Wayback 20120502183935

December 13, 2011 1 Comment

This post continues with the goal of binding a collection of elements to a Grid, with each object’s Row and Column properties indicating where in the Grid it should be located.  Last time we set the ItemsPanel property of an ItemsControl to order things in an 8×8 grid.  We also set the ItemTemplate property to include a Label with attached Grid.Row and Grid.Column properties.

In this post, we create a series of ChessPiece elements at runtime, representing a series of chess pieces.

ChessPieces = new ObservableCollection<ChessPiece>();
ChessPieces.Add(new ChessPiece("QR-Blk", class="num">1, class="num">1));
ChessPieces.Add(new ChessPiece("QN-Blk", class="num">1, class="num">2));
ChessPieces.Add(new ChessPiece("QB-Blk", class="num">1, class="num">3));
ChessPieces.Add(new ChessPiece("Q-Blk", class="num">1, class="num">4));
ChessPieces.Add(new ChessPiece("K-Blk", class="num">1, class="num">5));
ChessPieces.Add(new ChessPiece("KB-Blk", class="num">1, class="num">6));
ChessPieces.Add(new ChessPiece("KN-Blk", class="num">1, class="num">7));
ChessPieces.Add(new ChessPiece("KR-Blk", class="num">1, class="num">8));
ChessPieces.Add(new ChessPiece("P1-Blk", class="num">2, class="num">1));
ChessPieces.Add(new ChessPiece("P2-Blk", class="num">2, class="num">2));
ChessPieces.Add(new ChessPiece("P3-Blk", class="num">2, class="num">3));
ChessPieces.Add(new ChessPiece("P4-Blk", class="num">2, class="num">4));
ChessPieces.Add(new ChessPiece("P5-Blk", class="num">2, class="num">5));
ChessPieces.Add(new ChessPiece("P6-Blk", class="num">2, class="num">6));
ChessPieces.Add(new ChessPiece("P7-Blk", class="num">2, class="num">7));
ChessPieces.Add(new ChessPiece("P8-Blk", class="num">2, class="num">8));
OnPropertyChanged("ChessPieces");
