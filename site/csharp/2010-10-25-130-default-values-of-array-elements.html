#130 – Default Values of Array Elements

2010-10-25 | Original URL | Wayback 20130817191639

October 25, 2010 2 Comments

After declaring and instantiating an array, all of its elements take on default values.  The default value for all numeric types is 0 and for all reference types is null.  For enum types, the default value will be the element that has a value of 0–typically the first item listed in the enum.

float[] nums = new float[class="num">4];
float f1 = nums[class="num">0];    // 0.0
Cat[] pack = new Cat[class="num">5];
Cat c1 = pack[class="num">0];      // null
Days[] someDays = new Days[class="num">4];
Days thatDay = someDays[class="num">2];   // Sunday (1st in list)

The same rule applies for default values in a multi-dimensional array.  Each element is 0-valued or null.

int[,] nums = new int[class="num">2, class="num">3];
int aNum = nums[class="num">1, class="num">2];          // 0

With jagged arrays, you typically only instantiate the first dimension.  Elements of that dimension are null.

int[][] jagged = new int[class="num">3][];
object o = jagged[class="num">0];    // null
int n1 = jagged[class="num">0][class="num">0];   // Error: NullReferenceException
jagged[class="num">0] = new int[class="num">10];
int n2 = jagged[class="num">0][class="num">2];    // 0
